# 🏠 Village Address Directory App

A Flutter-based mobile application developed to digitally manage and organize village resident address records. The application provides an easy-to-use interface for storing, searching, updating, and viewing resident information, making record management more efficient and accessible.

## 📱 Features

* Add new resident records
* View resident details
* Update existing records
* Delete records
* Search residents by name or address
* Responsive and user-friendly UI
* Local data storage using SQLite
* Google Maps integration for location tracking
* Fast and efficient navigation

## 🛠️ Technologies Used

* Flutter
* Dart
* SQLite
* Google Maps API
* Android Studio
* Git & GitHub

## 🚀 Getting Started

### Prerequisites

* Flutter SDK
* Android Studio / VS Code
* Dart SDK
* Android Emulator or Physical Device

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/village-address-directory.git
```

2. Navigate to the project directory

```bash
cd village-address-directory
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
